Installation

Get Ryx on your machine

Ryx installs in seconds. Available for Linux, macOS, and Windows. Requires no additional dependencies for basic use.

1

Install via shell script

Run this command in your terminal. It will download and install the latest Ryx release.

curl -fsSL https://ryx.dev/install.sh | sh
2

Reload your shell

Add Ryx to your PATH by restarting your terminal or running:

source ~/.bashrc # or ~/.zshrc
3

Verify installation

Confirm Ryx is installed correctly:

ryx --version
4

Run your first program

Create hello.ryx and run it:

echo 'act main() { println("Hello, Ryx!") }' > hello.ryx
ryx run hello.ryx
1

Install with Homebrew (recommended)

If you have Homebrew installed:

brew install ryx
2

Alternative: shell installer

Or use the universal installer:

curl -fsSL https://ryx.dev/install.sh | sh
3

Verify & run

ryx --version
ryx run hello.ryx
1

Install via winget

Open PowerShell and run:

winget install ryx-lang.ryx
2

Or download the installer

Download the .msi installer from the GitHub releases page and run it.

Download .msi →
3

Verify in PowerShell

ryx --version
1

Clone the repository

git clone https://github.com/razen-core/ryx.git
2

Build from source

cd ryx
cargo build --release

⚠️ Requires Rust toolchain (cargo). The Ryx compiler frontend is written in Rust.

System Requirements

Operating SystemLinux (x86_64, ARM64), macOS 12+, Windows 10+
Architecturex86_64, ARM64 / Apple Silicon
C Compiler (runtime)GCC 11+ or Clang 14+ (for ryx build)
TCC (dev mode)Auto-installed — used by ryx run
Disk Space~50 MB for the Ryx toolchain
